# Edit cross-tool Global Agent Instructions

Carrent's personalization setting will edit the cross-tool `~/.agents/AGENTS.md` file rather than runtime-specific files such as `~/.kimi-code/AGENTS.md`. Kimi remains the primary runtime, but Carrent is an Agent GUI rather than a Kimi-only settings editor, so the visible "global instructions" surface should represent the user's shared agent preferences across compatible tools.
